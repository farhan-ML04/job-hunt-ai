# 🚀 Job Hunt AI

Built an **AI-powered job application assistant** that automates resume tailoring and job matching using LLMs.

---

## 💡 What I Built

* End-to-end pipeline to **analyze job descriptions and resumes**
* Automated **resume optimization & cover letter generation**
* Integrated **multiple AI agents** for modular workflow
* Deployed a **live Streamlit app on HuggingFace**

---

## 🛠 Tech Used

* Python, Streamlit
* OpenAI API (LLM)
* REST APIs (USAJobs, Google)
* Docker (for deployment)

---

## 📈 What It Solves

* Reduces manual effort in job applications
* Improves resume-job matching
* Generates ready-to-use application content
